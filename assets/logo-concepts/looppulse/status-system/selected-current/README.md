# LoopPulse Status Mascot Set

This folder records the currently selected six mascot states for in-app status expression.

Important boundary:
- These images are not native macOS menu bar icons.
- Do not crop or downscale these raster mascot images into `src-tauri/icons/tray-*.png`.
- The native tray icon should be designed separately as a 22pt macOS-friendly symbol/template asset.
- Mascot images can be used in the panel, detail views, onboarding, notifications, and design review boards.

Current selected states:
- `01-normal`: alive / idle
- `02-working`: working
- `03-waiting`: waiting for confirmation
- `04-warning`: warning / attention
- `05-error`: error / needs handling
- `06-offline`: stopped / offline
