# LoopPulse 内测版安装说明（macOS）

> 适用对象：第一批内测用户
> 适用版本：LoopPulse 内测版（**未签名 / 未公证**）
> 你的系统：macOS 12 (Monterey) 及以上，Apple Silicon (M 系列芯片)

---

## 这是什么

LoopPulse 是一个常驻 **菜单栏** 的小工具，用来实时观察你本机正在运行的 AI 编码 Agent（Claude Code / Codex / OpenCode）的状态：在干活、卡住、报错、还是快到额度上限。

它**只读取本机数据，不上传任何内容**（代码、对话、密钥都不会外传）。

---

## 重要：为什么打开时会被系统拦下

这个内测版**没有经过 Apple 付费签名和公证**（正式版才会做）。所以 macOS 的 Gatekeeper 第一次打开时会拦截，提示类似：

> “LoopPulse”已损坏，无法打开。 / 无法打开“LoopPulse”，因为 Apple 无法检查其是否包含恶意软件。

**这是未签名 App 的正常现象，不代表 App 真的有问题。** 按下面任一方法放行即可。

---

## 安装步骤

### 1. 拷贝到「应用程序」

打开下载到的 `LoopPulse_0.1.0_aarch64.dmg`，把里面的 **LoopPulse** 图标拖进 **「应用程序」(Applications)** 文件夹。

### 2. 首次打开（二选一）

**方法 A —— 右键打开（最简单，推荐先试）**

1. 在「应用程序」里找到 LoopPulse；
2. **按住 Control 键点击**（或右键点击）图标 → 选 **「打开」**；
3. 弹窗里再点一次 **「打开」**。

> 如果方法 A 弹出的是「已损坏」且只有「移到废纸篓」按钮、没有「打开」，请用方法 B。

**方法 B —— 终端移除隔离属性（方法 A 不行时用）**

1. 打开「终端」(Terminal)；
2. 粘贴下面这行命令，回车（会要求输入开机密码，输入时不显示字符，正常）：

   ```bash
   xattr -dr com.apple.quarantine /Applications/LoopPulse.app
   ```

3. 然后正常双击打开 LoopPulse。

> 这行命令的作用：去掉「从网络下载」的隔离标记，让未签名 App 能直接运行。只对 LoopPulse 这一个 App 生效。

### 3. 看菜单栏

打开后**主界面不会出现在 Dock 或窗口**——LoopPulse 是菜单栏工具。看屏幕**右上角菜单栏**，会多出一个小图标。**点它**就会从右侧滑出面板。

---

## 首次使用提示

- **没有会话时**面板会显示空态，属正常——启动一个 Claude Code / Codex 会话后会自动出现。
- 想开桌面通知：点面板左下角设置 → 打开通知开关，系统会请求一次通知权限，**请允许**。
- 想看完整工作台：面板里有进入「完整视图」的入口。
- 想再看一遍引导：设置里有「重新查看引导」。

---

## 卸载

把 `/Applications/LoopPulse.app` 拖到废纸篓即可。配置和本地历史存在：

```text
~/Library/Application Support/observer/
```

如需彻底清除，一并删除该目录。

内测最重要的就是你的真实反馈。遇到以下情况请记下来告诉我们：

- **状态判断不准**：某个会话明明在跑/卡住，LoopPulse 显示错了；

- **任何"我以为它会 X，结果它 Y"** 的预期落差。
